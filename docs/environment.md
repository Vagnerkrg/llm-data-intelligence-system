# Environment

## Project Environment

Project:
LLM Data Intelligence System

3.12

venv

./venv


## Development Environment

Operating System:
Windows

Shell:
PowerShell


## Core Dependencies

The project uses:

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter
- LlamaIndex
- Transformers
- Embeddings libraries
- LLM API clients


## Virtual Environment

### Activate Environment

Windows PowerShell:

```powershell
.\venv\Scripts\Activate.ps1
```

### Deactivate Environment

```powershell
deactivate
```


## Installation

Install dependencies:

```powershell
pip install -r requirements.txt
```


## Verification

Check Python version:

```powershell
python --version
```

Check installed packages:

```powershell
pip list
```


## Project Philosophy

The environment should remain stable.

Existing dependencies should not be removed without technical justification.

Changes must be documented before modifying the project foundation.
