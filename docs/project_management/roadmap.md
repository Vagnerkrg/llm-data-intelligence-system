# Project Roadmap

## Objective

Provide the consolidated roadmap for the current platform evolution.

## Strategic direction

The project progresses from retrieval and analysis capabilities toward a more autonomous, agent-based, and self-improving intelligence platform.

- V1.8: agent intelligence expansion
- V1.9: agent platform expansion
- V1.10: multi-tool intelligence
- V1.11: agent reasoning expansion
- V1.12: goal-driven planning
- V1.13: execution lifecycle architecture
- V1.14: cognitive decision layer
- V1.15: decision intelligence pipeline
- V1.16: autonomy and memory evolution
- V1.17: self-improvement expansion
