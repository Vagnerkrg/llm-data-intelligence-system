# Architecture Principles

## Objective

Preserve the core architectural principles in a canonical document that can be referenced by the broader documentation set.

## Principles

- Platform first, technology second
- Modular architecture
- Separation of responsibilities
- Dependency direction toward core abstractions
- Replaceable components
- Documentation as part of the software
- Continuous evolution
- Reusable knowledge
- Capability-oriented development
- Quality over speed
